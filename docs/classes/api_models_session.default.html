    • Defined in api/models/Session.ts:71

    Creates a session model with the specified parameters. Note this should only be done once per instance. Ensure initialization is successful before executing certain functions within this class, otherwise errors may be encountered.

    Parameters

    • apiClient: default

      Client configured to communicate with Senseye's API.

    • userId: string

      Custom username or ID of the participant. Must be unique to the participant within the context of the API token/key of the client.

    • Optional surveyId: string

      ID of a Survey to associate to the session. Will ideally be the demographic survey pertaining to the partcipant.

    Returns Promise<{}>

    A Promise that will produce the created session's metadata.
